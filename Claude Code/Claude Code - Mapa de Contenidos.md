# 🧭 Claude Code — Mapa de Contenidos

Este vault es una guía progresiva para entender **Claude Code** desde cero hasta un nivel avanzado: no solo cómo usarlo, sino cómo piensa, cómo está construido conceptualmente y cómo colaborar mejor con él. Se construye capítulo a capítulo, en el mismo orden en que se explica en la conversación que lo originó — cada nota nueva se enlaza con las anteriores en lugar de repetir contenido.

> [!tip] Cómo usar este vault
> - Abre esta carpeta completa en Obsidian como un **vault** (`Archivo → Abrir carpeta como vault`).
> - Empieza por el [[Claude Code Capitulo 01 - Fundamentos y Filosofia|Capítulo 1]] y avanza en orden — cada capítulo se apoya en el anterior.
> - Usa la **vista de grafo** de Obsidian para ver visualmente cómo se conectan los conceptos.
> - Los capítulos marcados ⏳ son marcadores de posición: ya existen como notas (para que los enlaces no queden rotos) pero su contenido se irá completando a medida que avancemos.
> - Cuando termines toda la guía, usa el [[Mapa Mental]] como repaso general y el [[Plan de Estudio 7 Dias]] para consolidar lo aprendido con práctica real.

## Encuadre epistemológico de todo el vault

A lo largo de esta guía, todo el contenido técnico se etiqueta con una de estas categorías (definidas con detalle en el [[Claude Code Capitulo 01 - Fundamentos y Filosofia#1.6 Información pública vs. simplificación pedagógica|§1.6 del Capítulo 1]]):

- 🔵 **Documentado oficialmente** por Anthropic (se cita la fuente cuando es posible).
- 🟢 **Observable directamente** en una sesión real de Claude Code (evidencia de primera mano, no de documentación).
- 🟡 **Simplificación pedagógica** — un modelo mental útil para entender el comportamiento, pero no una descripción literal del mecanismo interno.
- ⚪ **No público / desconocido** — se indica explícitamente en vez de especular.

Esta distinción es la columna vertebral de toda la guía: entender Claude Code bien significa también saber qué partes de su funcionamiento son verificables y cuáles son simplificaciones útiles.
